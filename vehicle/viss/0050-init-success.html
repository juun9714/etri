[0050: establish a WebSocket session with 'wss' shema]

Test assertion:
When the client makes a request to the server to create a new WebSocket session
whose schema and sub-protocol are 'wss' and 'wvss' with a version number suffix,
a WebSocket session is created.

Test Scenario:
1. Try to open websocket connection with 'wss' protocol + 'wvss1.0" subprotocol
2. Confirm the websocket connection will be established.

Expected Result:
- Websocket connection is established.
